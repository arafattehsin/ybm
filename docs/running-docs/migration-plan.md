## 1. Executive Summary

### Project Goal

Migrate YUM by Maryam from WordPress/WooCommerce on Bluehost to a modern Next.js application hosted on Azure Static Web Apps, maintaining the exact same user experience and visual design.

### Key Benefits

| Aspect           | Before (WordPress) | After (Next.js + Azure) |
| ---------------- | ------------------ | ----------------------- |
| Monthly Cost     | $15-30             | $0                      |
| Page Load        | 2-4 seconds        | <1 second               |
| Maintenance      | 8-14 hrs/month     | 1-2 hrs/month           |
| Security Patches | Constant           | Minimal                 |
| Transaction Fees | 1.7% (Stripe)      | 1.7% (Stripe)           |

### Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **State Management:** Zustand (cart)
- **Hosting:** Azure Static Web Apps (Free tier)
- **Payments:** Stripe Checkout
- **Email:** SendGrid / Resend (free tier)
- **Data:** JSON files (no database)

2. API creates Stripe Checkout Session with all cart items
3. User redirected to Stripe's hosted checkout page
4. Stripe handles:
   - Customer info collection
   - Shipping/pickup address (optional)
   - Payment processing
   - 3D Secure authentication
5. Success â†’ Redirect to `/success?session_id=xxx`
6. Webhook â†’ Send order notification email

## 7. API Endpoints

### 7.1 Stripe Checkout Session

**Endpoint:** `POST /api/checkout`

**Request:**

```typescript
interface CheckoutRequest {
  items: {
    productId: string;
    productName: string;
    size: string;
    addons: string[];
    quantity: number;
    unitPrice: number; // in cents
  }[];
  customerEmail?: string;
}
```

**Response:**

```typescript
interface CheckoutResponse {
  sessionId: string;
  url: string; // Stripe Checkout URL
}
```

**Implementation:**

```typescript
// src/app/api/checkout/route.ts
import Stripe from "stripe";
import { NextResponse } from "next/server";

const stripe = new Stripe(process.env.STRIPE_SECRET_KEY!);

export async function POST(request: Request) {
  const { items } = await request.json();

  const lineItems = items.map((item) => ({
    price_data: {
      currency: "aud",
      product_data: {
        name: `${item.productName} (${item.size})`,
        description:
          item.addons.length > 0
            ? `Add-ons: ${item.addons.join(", ")}`
            : undefined,
      },
      unit_amount: item.unitPrice,
    },
    quantity: item.quantity,
  }));

  const session = await stripe.checkout.sessions.create({
    mode: "payment",
    line_items: lineItems,
    success_url: `${process.env.NEXT_PUBLIC_BASE_URL}/success?session_id={CHECKOUT_SESSION_ID}`,
    cancel_url: `${process.env.NEXT_PUBLIC_BASE_URL}/cart`,
    shipping_address_collection: {
      allowed_countries: ["AU"],
    },
  });

  return NextResponse.json({
    sessionId: session.id,
    url: session.url,
  });
}
```

### 7.2 Stripe Webhook

**Endpoint:** `POST /api/webhook`

**Purpose:** Receive Stripe events, send order notification email

**Implementation:**

```typescript
// src/app/api/webhook/route.ts
import Stripe from "stripe";
import { NextResponse } from "next/server";

const stripe = new Stripe(process.env.STRIPE_SECRET_KEY!);

export async function POST(request: Request) {
  const payload = await request.text();
  const sig = request.headers.get("stripe-signature")!;

  let event: Stripe.Event;

  try {
    event = stripe.webhooks.constructEvent(
      payload,
      sig,
      process.env.STRIPE_WEBHOOK_SECRET!
    );
  } catch (err) {
    return NextResponse.json({ error: "Webhook error" }, { status: 400 });
  }

  if (event.type === "checkout.session.completed") {
    const session = event.data.object as Stripe.Checkout.Session;

    // Send order notification email
    await sendOrderNotification({
      customerEmail: session.customer_email,
      customerName: session.customer_details?.name,
      shippingAddress: session.shipping_details?.address,
      amount: session.amount_total,
      sessionId: session.id,
    });
  }

  return NextResponse.json({ received: true });
}
```
